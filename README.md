# Kim jestem?

Jako absolwent kierunku Big Data na SGH rozwijam się w obszarze Data Science. W trakcie studiów zdobyłem doświadczenie w rozwiązywaniu złożonych problemów analitycznych i budowie modeli predykcyjnych.
